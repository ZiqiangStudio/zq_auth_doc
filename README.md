# 自强 Auth 文档

## 开始

```bash
git clone https://github.com/ZiqiangStudio/zq_auth_doc
cd zq_auth_doc
yarn install
```

## 常用命令

- `commit` 使用 commitizen 提交代码
- `dev` 在本地运行项目
- `build` 打包项目至 `dist`
- `preview` 预览 `dist` 内打包后资源
- `lint` 检查代码是否符合规范
- `lint --fix` 检查代码是否符合规范，并自动修复
- `typecheck` 检查 TypeScript 代码是否类型安全
